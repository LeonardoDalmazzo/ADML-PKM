1. Estrutura Básica e Tipos Primitivos

Estrutura "Hello World"

#include <stdio.h> // Biblioteca de Entrada/Saída
#include <locale.h> // Para acentuação

int main() {
    setlocale(LC_ALL, "Portuguese");
    printf("Olá, Mundo!\n");
    return 0;
}

Tipos Primitivos & Formatadores

Tipo Descrição Formatador (printf/scanf)
int Números inteiros %d ou %i
float Decimais (precisão simples) %f
double Decimais (precisão dupla) %lf
char Um único caractere %c
char[] String (Array de chars) %s
void Vazio (sem retorno) -

Exemplos extraídos do projeto Hortifruti.

Switch Case (Menu)
switch (opcao) {
    case 1:
        senhaCaixa();
        break;
    case 2:
        senhaGerente();
        break;
    default:
        printf("Opcao invalida.\n");
        break;
}
Do-While (Repetição)
do {
    // Mostra menu...
    scanf("%d", &opcao);
    // Executa switch...
} while (opcao != 5); 
// Repete enquanto não escolher sair

4. Criando Funções

Funções modularizam o código. No seu projeto, cadastrarProduto() é um ótimo exemplo de função void (sem retorno).

// Declaração (Exemplo simples)
float calcularTotal(float preco, int qtd) {
    return preco * qtd;
}

// Uso no main
float total = calcularTotal(10.50, 2);
Dica: Em C, se a função for criada depois do main, você precisa declarar o protótipo dela antes.

5. Ponteiros, Memória e Conversão

Ponteiros (*) e Endereço (&)

Variáveis que armazenam o endereço de memória de outra variável.

int numero = 10;
int *ponteiro = № // Aponta para o endereço de 'numero'

// No seu código (scanf precisa do endereço):
scanf("%d", &opcao); 

// Strings (Vetores) já são ponteiros:
char nome[50];
scanf("%s", nome); // Sem '&'

Cast (Conversão de Tipo)

Forçar um tipo a se comportar como outro.

int a = 10, b = 3;
// Sem cast, divisão de int dá int (3)
float resultado = (float)a / b; 
// Resultado: 3.333...

6. Structs (Estruturas de Dados)

Structs permitem criar seus próprios tipos de dados complexos. O melhor exemplo é o seu Produto:

hortiFrutiLib.h
typedef struct {
    int id;
    char nome[100];
    char tipo[100];
    int quantidade;
    float preco;
} Produto;

// Criando um array de Structs (Banco de dados em memória)
Produto produto[500];

Para acessar os dados, usamos o ponto: produto[i].nome.

7. Manipulação de Arquivos (FILE)

Persistência de dados. Modos comuns: "r" (leitura), "w" (escrita - sobrescreve), "a" (append - adiciona).

Exemplo: salvarProdutos()
FILE *arquivo = fopen("hortiFruti.txt", "w"); // Abre para escrita

if (arquivo == NULL) {
    printf("Erro ao abrir arquivo!");
} else {
    // Escreve no arquivo formatado
    fprintf(arquivo, "%d;%s;%.2f\n", id, nome, preco);
    fclose(arquivo); // Importante fechar!
}

8. Modularização e Bibliotecas

Libs Nativas

  • <stdio.h>: Entrada e Saída (printf, scanf, FILE).
  • <stdlib.h>: Gestão de memória, conversões (atoi, atof), system().
  • <string.h>: Manipulação de texto (strcpy, strcmp, strlen).
  • <math.h>: Funções matemáticas (pow, sqrt).
  • <locale.h>: Configuração de idioma/acentos.

Suas Libs (.h) e Header Guards

Como evitar que uma biblioteca seja importada duas vezes? Usando Header Guards.

hortiFrutiLib.h
#ifndef HORTIFRUTILIB_H  // Se não definido...
#define HORTIFRUTILIB_H  // Defina agora.

// Todo seu código, structs e funções aqui...

#endif // Fim da definição

Para usar no main: #include "suaLib.h" (aspas para arquivos locais).
