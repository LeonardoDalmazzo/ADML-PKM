Atalhos de Produtividade

Navegação e Seleção

  • CTRL + Setas Ir para o final dos dados
  • CTRL + SHIFT + Setas Selecionar até o final
  • CTRL + Page Up/Down Trocar de aba
  • CTRL + Espaço Selecionar Coluna inteira
  • SHIFT + Espaço Selecionar Linha inteira

Ações Rápidas

  • ALT + = Autossoma (Soma rápida)
  • CTRL + ; Inserir Data atual
  • CTRL + E Preenchimento Relâmpago
  • CTRL + 1 Menu Formatar Células
  • CTRL + T Criar Tabela Oficial
Dica (Ctrl+E): Digite um exemplo do que você quer extrair (ex: o primeiro nome de uma lista completa) na coluna ao lado e aperte Ctrl+E. O Excel completa o resto sozinho!

Fórmulas do Dia a Dia

Função Sintaxe Básica Para que serve
SE (IF) =SE(teste; verdadeiro; falso) Tomada de decisão lógica.
SOMASES =SOMASES(intervalo_soma; critérios...) Soma condicional (ex: Vendas do mês X).
TEXTO =TEXTO(A1; "dd/mm") Formata números/datas para texto.
ARRUMAR =ARRUMAR(A1) Remove espaços extras indesejados.

A Evolução: PROCV vs PROCX

Antigo: PROCV (VLOOKUP)

Busca apenas à direita, lento e quebra se inserir colunas.

=PROCV(valor; matriz; num_coluna; 0)

Moderno: PROCX (XLOOKUP)

Busca em qualquer direção, mais rápido e com tratamento de erro nativo.

=PROCX(valor; onde_buscar; o_que_retornar; "Não achei")

Tabela Dinâmica

O que é?

A ferramenta mais poderosa para resumir grandes volumes de dados sem fórmulas.

  • Arrastar e Soltar campos.
  • Agrupar datas (Meses/Anos) automaticamente.
  • Criar Segmentação de Dados (Botões de filtro).

Atalhos e Dicas

  • ALT + T + B Criar Tabela Dinâmica
  • Clique duas vezes em um número da tabela dinâmica para ver os detalhes (Drill down).

Power Query (ETL)

Esqueça o VBA para limpeza de dados. Use o Power Query para:

  • Importar dados de pastas, Web, PDF ou CSV.
  • Tratar colunas (Dividir, Substituir, Tipar).
  • Mesclar tabelas (VLOOKUP visual).
  • Tudo gravado em passos automáticos!
Acesse via: Dados > Obter Dados.

VBA (Macros)

Para automações que o Power Query não resolve. Use ALT + F11 para abrir.

Exemplo: Limpar células

Module1.bas
Sub LimparDados()
    ' Seleciona a aba e limpa intervalo
    Sheets("Planilha1").Range("A2:C100").ClearContents
    
    MsgBox "Dados limpos com sucesso!", vbInformation
End Sub
