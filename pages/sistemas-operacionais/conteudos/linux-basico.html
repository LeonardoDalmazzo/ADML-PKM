Linux Basico

Conceitos fundamentais e estrutura do sistema.

Estrutura de diretorios (FHS)

No Linux tudo comeca em /, sem letras de drive.

/etc  -> configuracoes
/home -> usuarios
/var  -> logs e dados variaveis
/bin  -> executaveis essenciais

Sudo e root

sudo executa comandos administrativos de forma controlada, evitando uso continuo da conta root.

Navegacao no terminal

pwd
ls -la
cd /etc
cd ~
