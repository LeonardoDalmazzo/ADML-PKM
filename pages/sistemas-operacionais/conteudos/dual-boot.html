Dual boot Windows + Linux

Dual boot permite usar Windows e Linux no mesmo computador. O segredo é planejamento, backup e seguir a ordem correta.

Antes de começar

  • Backup dos arquivos importantes
  • Verifique se o sistema usa UEFI + GPT
  • Tenha um pendrive bootável do Linux

Planejamento de partições

Diminuir partição do Windows

  1. Abra o Gerenciamento de Disco
  2. Selecione a partição C:
  3. Use Diminuir volume

Tamanho recomendado

  • Uso leve: 30–50 GB
  • Uso médio/dev: 80–150 GB

Ordem correta de instalação

Sempre instale o Windows primeiro e o Linux depois.

Instalando o Linux

Instalar ao lado do Windows

Se disponível, use a opção automática do instalador.

Bootloader (GRUB)

O GRUB permitirá escolher qual sistema iniciar.

Após a instalação

Atualize o Linux

  • Atualize pacotes
  • Instale drivers proprietários se necessário

Verifique a ordem de boot

  • Confira se o GRUB está como padrão

Problemas comuns

GRUB sumiu

  • Verifique a ordem no UEFI
  • Use um live USB para recuperar

Boas práticas

  • Evite misturar Legacy e UEFI
  • Tenha sempre mídia de recuperação
