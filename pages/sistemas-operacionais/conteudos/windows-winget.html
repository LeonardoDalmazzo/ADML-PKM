Windows e Winget

Atalhos de sistema e automacao com gerenciador de pacotes.

Sistema e essenciais

Basicos

  • Win abrir iniciar
  • Win + E explorador
  • Win + I configuracoes

Admin

  • Win + X menu rapido
  • Win + R executar
  • Ctrl + Shift + Esc task manager

Winget CLI

winget search "visual studio code"
winget install Microsoft.VisualStudioCode
winget upgrade --all
winget list
winget uninstall "nome do programa"

Gerenciamento de janelas

Use Win + setas para snap e Alt + Tab para alternancia rapida.

Desktops virtuais

Win + Ctrl + D cria novo desktop e Win + Ctrl + setas alterna.
