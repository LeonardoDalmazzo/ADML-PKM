IP Estatico e DNS no Windows

Configuracao manual de TCP/IPv4 com contexto de rede.

DHCP vs IP estatico

DHCP

Endereco automatico entregue pelo roteador.

Estatico

Endereco fixo para servidor, impressora ou regra de encaminhamento.

Passo a passo

  1. Abrir ncpa.cpl.
  2. Propriedades do adaptador -> IPv4.
  3. Definir IP, mascara e gateway.
  4. Configurar DNS preferencial e alternativo.

Exemplo de enderecamento

IP:      192.168.1.150
Mascara: 255.255.255.0
Gateway: 192.168.1.1
DNS:     8.8.8.8 / 8.8.4.4

Entendendo os campos

IP

Identidade do host na rede local.

Mascara

Define qual parte e rede e qual parte e host.

Gateway

Saida para redes externas.

DNS

Traducao de dominio para IP.
