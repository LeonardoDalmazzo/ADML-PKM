Google Sheets

Planilhas em nuvem com colaboracao, QUERY e Apps Script.

Atalhos web

  • Ctrl + / ver atalhos
  • Ctrl + Shift + V colar apenas valores
  • Ctrl + K inserir link
Dica: use sheet.new para criar planilha instantaneamente.

Navegacao e selecao

Movimento

Ctrl + setas e Ctrl + Home.

Selecao

Ctrl + Shift + setas para blocos.

Formatacao e estrutura

Estilo

Negrito, limpar formato e tachado.

Linhas e colunas

Inserir/excluir e preencher para baixo.

Formulas basicas

FormulaAcao
=SOMA()Soma intervalo
=SE()Condicional
=PROCV()Busca classica

Validacao e dados

Checkbox

TRUE/FALSE para fluxos de checklist.

Dropdown

Padronizacao de entrada e menos erro de digitacao.

Funcoes exclusivas

IMPORTRANGE

Importar dados de outra planilha.

FILTER

Filtrar dados com condicoes.

SPARKLINE

Mini graficos em celula.

GOOGLEFINANCE

Cotacoes em tempo real.

Funcao QUERY

=QUERY(A1:D100; "SELECT A, B WHERE C > 100 ORDER BY D DESC LIMIT 5")

Regex e limpeza

=REGEXMATCH(A1; "@gmail")
=REGEXEXTRACT(A1; "[0-9]+")
=REGEXREPLACE(A1; "[^0-9]"; "")

Apps Script

function onOpen() {
  SpreadsheetApp.getUi()
    .createMenu('Minhas Tools')
    .addItem('Rodar Script', 'minhaFuncao')
    .addToUi();
}
