Formatacao profissional, estilos e produtividade academica.

Formatacao agil

Texto essencial

  • Ctrl + N negrito
  • Ctrl + I italico
  • Ctrl + S sublinhado
  • Shift + F3 alternar caixa

Paragrafos

  • Ctrl + J justificar
  • Ctrl + E centralizar
  • Ctrl + L alinhar a esquerda

Navegacao e selecao

  • Ctrl + F localizar
  • Ctrl + G ir para
  • Ctrl + Home inicio do documento
  • F8 expandir selecao
  • Alt + drag selecao vertical

Estilos e sumario

Use estilos para gerar sumario automatico e manter consistencia de documento.

  • Ctrl + Alt + 1 titulo 1
  • Ctrl + Alt + 2 titulo 2
  • Ctrl + Shift + N normal

Academico e revisao

Quebras e referencias

  • Ctrl + Enter quebra de pagina
  • Alt + Ctrl + F nota de rodape

Track changes

  • Ctrl + Shift + E controlar alteracoes
  • F7 ortografia e gramatica

Imagens e tabelas

Prefira quebra de texto em quadrado/atraves para posicionar imagens sem quebrar fluxo.

Em tabelas, use Tab para navegar entre celulas.

Recursos modernos

Ditado

Win + H para voz no Windows.

Editor

Use painel Editor para clareza, concisao e linguagem formal.
