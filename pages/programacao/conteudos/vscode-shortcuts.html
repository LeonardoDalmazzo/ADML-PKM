VS Code Shortcuts

Atalhos essenciais para produtividade, navegacao e fluxo de desenvolvimento.

Edicao de codigo

Linhas

  • Alt + Up/Down Mover linha
  • Shift + Alt + Up/Down Duplicar linha
  • Ctrl + Shift + K Deletar linha
  • Ctrl + A Selecionar tudo

Selecao e comentarios

  • Shift + Alt + A Comentar bloco
  • Ctrl + D Proxima ocorrencia
  • Ctrl + Shift + L Todas as ocorrencias
  • Alt + Click Cursor multiplo

Navegacao e busca

Dica: Ctrl + Shift + P abre a Command Palette.
  • Ctrl + P Ir para arquivo
  • Ctrl + Shift + O Ir para simbolo
  • Ctrl + G Ir para linha
  • F12 Ir para definicao
  • Ctrl + Shift + F Buscar no projeto
  • Ctrl + F Buscar no arquivo
  • Ctrl + H Substituir
  • Alt + Left/Right Historico de navegacao

Janelas e layout

Editor

  • Ctrl + \ Split editor
  • Ctrl + 1 / 2 Focar grupo
  • Ctrl + W Fechar aba
  • Ctrl + K Z Modo Zen

Interface

  • Ctrl + B Sidebar
  • Ctrl + J Painel inferior
  • Ctrl + Tab Trocar abas
  • Ctrl + + / - Zoom

Terminal integrado

  • Ctrl + ' Abrir/fechar terminal
  • Ctrl + Shift + ' Novo terminal
  • Ctrl + L Limpar terminal

Markdown e extras

Preview

  • Ctrl + Shift + V Preview em aba
  • Ctrl + K V Preview lateral

Snippets uteis

  • ! + Tab gera estrutura HTML inicial.
  • lorem + Tab gera texto de preenchimento.
