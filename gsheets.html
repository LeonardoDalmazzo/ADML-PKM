Atalhos (Navegador)

  • CTRL + / Ver todos os atalhos
  • CTRL + SHIFT + V Colar apenas valores (Sem formatação)
  • CTRL + ALT + M Inserir comentário
  • CTRL + K Inserir Link
Dica: Digite doc.new, sheet.new ou slide.new na barra de endereço do navegador para criar um arquivo novo instantaneamente.

Funções Exclusivas do Google

IMPORTRANGE

Puxa dados de outra planilha do Google Sheets.

=IMPORTRANGE("URL_DA_PLANILHA"; "Aba!A1:B10")

SPARKLINE

Cria um mini gráfico dentro de uma única célula.

=SPARKLINE(A1:A10; {"charttype"\"bar"})

FILTER

Filtra uma tabela dinamicamente (retorna array).

=FILTER(A2:B10; C2:C10="Aprovado")

GOOGLEFINANCE

Cotações de moedas e ações em tempo real.

=GOOGLEFINANCE("USDBRL")

A Poderosa Função QUERY

Permite usar linguagem SQL dentro da planilha para manipular dados.

Exemplo Prático
=QUERY(A1:D100; "SELECT A, B WHERE C > 100 ORDER BY D DESC LIMIT 10")

Tradução: Selecione colunas A e B, onde a coluna C for maior que 100, ordene pela D (decrescente) e mostre só os top 10.

Google Apps Script (JS)

Automação baseada em JavaScript. (Extensões > Apps Script)

Exemplo: Criar Menu Personalizado

code.gs
function onOpen() {
  const ui = SpreadsheetApp.getUi();
  ui.createMenu('Minhas Ferramentas')
      .addItem('Enviar Email', 'enviarEmail')
      .addToUi();
}

function enviarEmail() {
  GmailApp.sendEmail("email@exemplo.com", "Assunto", "Corpo do email");
  Browser.msgBox("Email enviado!");
}
