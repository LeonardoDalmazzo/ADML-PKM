Atalhos (Navegador)

  • CTRL + / Ver todos os atalhos
  • CTRL + SHIFT + V Colar apenas valores (Sem formatação)
  • CTRL + ALT + M Inserir comentário
  • CTRL + K Inserir Link
Dica Ninja: Digite sheet.new na barra de endereço do navegador para criar uma planilha nova instantaneamente.

Navegação e Seleção

Movimento

  • CTRL + Setas Pular para o fim dos dados
  • CTRL + HOME Ir para célula A1
  • CTRL + PageUp/Down Alternar abas

Seleção

  • CTRL + SHIFT + Setas Selecionar bloco de dados
  • SHIFT + Espaço Selecionar linha inteira
  • CTRL + Espaço Selecionar coluna inteira

Formatação e Estrutura

Estilo

  • CTRL + B Negrito
  • CTRL + \ Limpar formatação
  • ALT + SHIFT + 5 Tachado (Risco)

Linhas e Colunas

  • CTRL + ALT + = Inserir linha/coluna
  • CTRL + ALT + - Excluir linha/coluna
  • CTRL + D Preencher abaixo (Fill Down)

Fórmulas Essenciais

Fórmula Ação

Validação e Interatividade

Checkboxes

Inserir > Caixa de Seleção.

  • Marcado = TRUE
  • Desmarcado = FALSE
  • Ótimo para usar dentro de fórmulas =SE().

Dropdowns (Menu Suspenso)

Dados > Validação de Dados.

Use para criar listas fixas ou puxar opções de um intervalo dinâmico, evitando erros de digitação.

Funções Exclusivas do Google

IMPORTRANGE

Puxa dados de outra planilha.

=IMPORTRANGE("URL"; "Aba!A:B")

SPARKLINE

Mini gráficos na célula.

=SPARKLINE(A1:A10; {"charttype"\"bar"})

FILTER

Filtra tabela (retorna array).

=FILTER(A:B; C:C="Sim")

GOOGLEFINANCE

Cotações em tempo real.

=GOOGLEFINANCE("USDBRL")

Função QUERY (SQL)

Permite usar linguagem SQL dentro da planilha.

Exemplo
=QUERY(A1:D100; "SELECT A, B WHERE C > 100 ORDER BY D DESC LIMIT 5")

Seleciona colunas A e B, onde C > 100, ordena por D e limita a 5 resultados.

Regex & Limpeza de Dados

Limpeza Básica

  • =ARRUMAR(A1) (TRIM) Remove espaços extras
  • =MAIÚSCULA(A1) (UPPER) Tudo em caixa alta
  • =PRI.MAIÚSCULA(A1) (PROPER) Nome Próprio

Expressões Regulares

Para extrações complexas.

=REGEXMATCH(A1; "@gmail") 
=REGEXEXTRACT(A1; "[0-9]+") 
=REGEXREPLACE(A1; "[^0-9]"; "")
