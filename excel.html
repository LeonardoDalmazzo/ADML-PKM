Atalhos de Produtividade

Navegação e Seleção

  • CTRL + Setas Ir para o final dos dados
  • CTRL + SHIFT + Setas Selecionar até o final
  • CTRL + Page Up/Down Trocar de aba
  • CTRL + Espaço Selecionar Coluna inteira
  • SHIFT + Espaço Selecionar Linha inteira

Ações Rápidas

  • ALT + = Autossoma (Soma rápida)
  • CTRL + ; Inserir Data atual
  • CTRL + 1 Menu Formatar Células
  • CTRL + T Criar Tabela (Table)
  • F4 Repetir última ação / Trancar célula ($)

Fórmulas do Dia a Dia

Função Sintaxe Descrição
SE (IF) =SE(teste; valor_se_v; valor_se_f) Lógica condicional básica.
SOMASES (SUMIFS) =SOMASES(intervalo_soma; critério1...) Soma com base em múltiplos critérios.
CONCAT =A1 & " " & B1 Juntar textos (ou usar fórmula CONCATENAR).
TEXTO =TEXTO(A1; "dd/mm/aaaa") Formata um número ou data como texto.

A Evolução: PROCV vs PROCX

Antigo: PROCV (VLOOKUP)

Busca apenas à direita e exige contar colunas.

=PROCV(valor_procurado; matriz; num_coluna; [0])

Moderno: PROCX (XLOOKUP)

Busca em qualquer direção, não quebra se inserir colunas e tem tratamento de erro.

=PROCX(valor_procurado; matriz_pesquisa; matriz_retorno; "Não achou")

VBA (Macros)

Use ALT + F11 para abrir o editor VBA.

Exemplo: Limpar células e avisar

VBA
Sub LimparDados()
    ' Seleciona a aba Planilha1
    Sheets("Planilha1").Select
    
    ' Limpa o conteúdo das células A2 até C100
    Range("A2:C100").ClearContents
    
    ' Exibe uma mensagem
    MsgBox "Dados limpos com sucesso!", vbInformation
End Sub
