Bloco de Notas vs. Editores

Programar no Bloco de Notas é possível, mas improdutivo. Editores de código (IDEs) como o VS Code oferecem ferramentas indispensáveis:

  • Syntax Highlighting: Cores diferentes para facilitar a leitura.
  • Emmet: Escreva HTML rápido (ex: ul>li*3 cria uma lista).
  • IntelliSense: O editor "adivinha" o que você quer digitar.
  • Terminal Integrado: Rode comandos sem sair da tela.

Dica de Instalação

Durante a instalação, marque estas opções para facilitar sua vida:

  • Adicionar "Abrir com Code" ao menu de contexto de arquivo.
  • Adicionar "Abrir com Code" ao menu de contexto de diretório.

Extensões Recomendadas

Acesse o menu lateral (ícone de quadrados) ou use CTRL + SHIFT + X.

Extensão Categoria Para que serve
Auto Complete Tag Produtividade Extension Pack (2 em 1): Renomeia e fecha tags HTML automaticamente.
Live Server Servidor Atualiza o navegador automaticamente ao salvar (Ctrl+S).
Prettier Formatação Formata o código e ajusta a indentação automaticamente.
vscode-icons Visual Adiciona ícones específicos para cada tipo de arquivo/pasta.
Better Comments Visual Comentários coloridos! Use ! para alerta, ? para dúvida, etc.
CodeSnap Social Tira prints bonitos do seu código para postar no LinkedIn/Instagram.
GitLens (ou GitHub PR) Git Superpoderes para o Git: vê quem editou a linha (blame), histórico e mais.
Auto Save: Habilitar essa opção no VS Code pode ser muito prático e uma boa solução junto ao Live Server

Snippet "!" (Estrutura Básica)

Não digite a estrutura manualmente! Crie um arquivo index.html, digite ! e aperte TAB.

O resultado gerado:

index.html
<!DOCTYPE html>
<html lang="pt-br"> <!-- Mude para pt-br -->
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <!-- Seu código aqui -->
</body>
</html>
