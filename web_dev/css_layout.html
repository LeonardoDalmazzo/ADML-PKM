Flexbox vs CSS Grid

Flexbox (1D)

Ótimo para alinhar itens em uma direção (linha OU coluna). Use para menus, centralização e componentes pequenos.

CSS Grid (2D)

Poderoso para duas direções (linhas E colunas) simultaneamente. Use para o layout principal da página (Header, Sidebar, Main, Footer).

Grid Básico

A propriedade mágica é display: grid.

.container {
    display: grid;
    /* Cria 3 colunas: 1ª 200px, 2ª Automática, 3ª 1 fração do espaço restante */
    grid-template-columns: 200px auto 1fr;
    
    /* Espaçamento */
    gap: 20px;
}
Unidade fr: 1fr significa "uma fração do espaço livre". É muito melhor que usar % em grids.

Grid Areas (Visual)

Você pode "desenhar" o layout no CSS nomeando as áreas.

.layout {
    display: grid;
    grid-template-areas:
        "header header header"
        "sidebar main main"
        "footer footer footer";
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

Responsividade (@media)

Media Queries permitem mudar o CSS dependendo do tamanho da tela.

/* Padrão (Desktop) */
.box { width: 50%; }

/* Se a tela for menor que 768px (Tablet/Celular) */
@media (max-width: 768px) {
    .box {
        width: 100%; /* Ocupa a tela toda */
        background-color: red;
    }
}

Conceito Mobile First

Em vez de fazer o site para Desktop e "espremer" para o celular, faça o CSS base pensando no celular (telas pequenas) e use min-width para expandir.

  • 1. Escreva o CSS normal (para celular).
  • 2. Adicione @media (min-width: 768px) para Tablets.
  • 3. Adicione @media (min-width: 1024px) para Desktop.

Isso torna o site mais leve e rápido em dispositivos móveis.
