Adeus var, olá Let & Const

O Problema do var

O antigo var "vaza" escopo (pode ser acessado fora de blocos if/for) e permite redeclaração, causando bugs silenciosos.

A Solução

  • const: Para valores que não vão mudar de referência (Ex: configurações, funções, imports). Use sempre que possível.
  • let: Para valores que vão mudar (Ex: contadores, inputs).

Arrow Functions

Uma sintaxe mais curta para escrever funções. Essencial para Callbacks e React.

Função Clássica
function somar(a, b) {
    return a + b;
}
Arrow Function
const somar = (a, b) => a + b;

// Se tiver 1 argumento, dispensa ()
const dobro = n => n * 2;

Sintaxe Sugar

Template Literals (``)

Permite interpolar variáveis sem ficar somando strings.

// Antigo
"Olá " + nome + "!";

// Novo (Crase)
`Olá ${nome}!`;

Desestruturação

Extrair dados de objetos/arrays rapidamente.

const user = { nome: 'Leo', idade: 25 };

// Cria variaveis 'nome' e 'idade'
const { nome, idade } = user;

console.log(nome); // 'Leo'

Map, Filter & Reduce

Pare de usar for loop para tudo! Use métodos funcionais para transformar dados.

Método O que faz Exemplo React
.map() Cria um novo array transformando cada item. Criar listas de componentes a partir de dados.
.filter() Cria um novo array apenas com itens que passam no teste. Remover um item de uma lista (To-do list).
.find() Retorna o primeiro item achado. Buscar um usuário pelo ID. 
const precos = [10, 20, 30];

// Dobra os valores
const dobro = precos.map(p => p * 2); 
// Resultado: [20, 40, 60]

Módulos (Import/Export)

Organize o código em arquivos separados.

arquivo.js

export const PI = 3.14;

export default function somar(a, b) {
    return a + b;
}

main.js

// Default (sem chaves)
import somar from './arquivo.js';

// Named (com chaves)
import { PI } from './arquivo.js';
