O que é o DOM?

DOM (Document Object Model) é a representação do seu HTML na memória do navegador. É uma "árvore" de elementos que o JavaScript pode acessar e modificar.

No JS, o ponto de entrada é o objeto global document.

Selecionando Elementos

Métodos Modernos

  • document.querySelector('.classe') Pega o primeiro elemento encontrado.
  • document.querySelectorAll('div') Pega todos (retorna uma lista).
  • document.getElementById('meuId') Pega elemento pelo ID (Rápido).

Exemplo

// HTML: <h1 id="titulo">Olá</h1>

const titulo = document.getElementById('titulo');
console.log(titulo); // Mostra o elemento no console

Alterando Conteúdo

Depois de selecionar, você pode mudar o que está dentro do elemento.

Propriedade O que faz Segurança
elemento.innerText Muda apenas o texto visível. Seguro ✅
elemento.innerHTML Interpreta HTML (cria tags). Cuidado (Risco XSS) ⚠️
elemento.value Pega valor de Inputs/Formulários. -

Manipulando CSS

Via Style (Inline)

Altera o estilo direto no elemento.

titulo.style.color = 'red';
titulo.style.backgroundColor = 'black';
// Note: usa camelCase (backgroundColor)

Via Classes (Recomendado)

Adiciona ou remove classes do CSS.

titulo.classList.add('ativo');
titulo.classList.remove('oculto');
titulo.classList.toggle('dark-mode');

Eventos (Interatividade)

O segredo da interatividade é "escutar" o que o usuário faz.

Exemplo: Botão que muda cor
const botao = document.querySelector('#meuBotao');
const corpo = document.body;

// Adiciona um "Ouvinte de Evento"
botao.addEventListener('click', function() {
    corpo.style.backgroundColor = 'blue';
    alert('Você clicou no botão!');
});
Outros eventos comuns: submit (formulários), input (digitação), scroll (rolagem da página).
