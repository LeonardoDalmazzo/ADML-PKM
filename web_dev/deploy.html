O que é Deploy?

Deploy é o processo de tirar o site do seu computador (localhost) e colocá-lo em um servidor público na internet para que qualquer pessoa possa acessar.

GitHub Pages

A opção mais fácil para sites estáticos (HTML/CSS/JS) que já estão no GitHub.

Passo a Passo

  1. Suba seu projeto para um repositório no GitHub.
  2. Vá em Settings (Configurações do repositório).
  3. No menu lateral, clique em Pages.
  4. Em "Branch", selecione main (ou master) e salve.

Resultado

Em alguns minutos, seu site estará disponível em:
seu-usuario.github.io/nome-do-repo

Vercel & Netlify

Plataformas modernas de hospedagem. São melhores que o GitHub Pages para projetos React ou que precisam de rotas mais complexas.

Vantagens

  • Deploy automático a cada git push.
  • HTTPS (cadeado de segurança) automático.
  • CDN global (site carrega rápido no mundo todo).

Como usar

  1. Crie conta na Vercel.com ou Netlify.com.
  2. Clique em "Add New Project".
  3. Conecte sua conta do GitHub.
  4. Selecione o repositório e clique em Deploy.
