Estrutura Semântica

Não use apenas div para tudo! Use tags que tenham significado para o navegador e leitores de tela (SEO & Acessibilidade).

Estrutura Correta
<header>Cabeçalho/Menu</header>
<main>
    <section>
        <h1>Título Principal</h1>
        <article>Conteúdo do Post</article>
    </section>
</main>
<footer>Rodapé</footer>

Tags Essenciais

  • <header>: Topo do site ou seção.
  • <nav>: Links de navegação.
  • <main>: Conteúdo principal (único por pág).
  • <section>: Agrupamento temático.
  • <article>: Conteúdo independente (post, card).
  • <aside>: Barra lateral ou info extra.

O CSS Box Model

Todo elemento HTML é uma caixa retangular composta por 4 camadas. Entender isso é vital para layouts.

Camada Descrição CSS
Content O conteúdo real (texto, img). width, height
Padding Espaço interno (entre conteúdo e borda). padding: 20px;
Border A borda em si. border: 1px solid #000;
Margin Espaço externo (empurra outros elementos). margin: 10px;
Dica de Ouro: Sempre use box-sizing: border-box; no seu CSS global. Isso faz com que o Padding e a Borda fiquem dentro da largura definida, evitando quebrar o layout.

Seletores CSS

Básicos

  • tag Seleciona todos (Ex: p, div)
  • .classe Seleciona grupo (Ex: .card)
  • #id Seleciona único (Ex: #menu)
  • * Seleciona tudo (Universal)

Avançados

  • :hover Ao passar o mouse
  • div > p Filho direto
  • nth-child(2) Segundo item da lista
  • ::before Cria elemento visual antes

Flexbox (Layout Moderno)

Esqueça float. Use Flexbox para alinhar coisas em uma dimensão (linha ou coluna).

CSS
.container {
    display: flex;              /* Ativa o Flexbox */
    justify-content: center;    /* Alinha na horizontal (Eixo principal) */
    align-items: center;        /* Alinha na vertical (Eixo cruzado) */
    gap: 20px;                  /* Espaço entre itens */
    flex-wrap: wrap;            /* Quebra linha se faltar espaço */
}

Unidades e Cores

Medidas

  • px: Pixels (Fixo). Bom para bordas.
  • %: Porcentagem (Relativo ao pai).
  • rem: Relativo à fonte raiz (Acessibilidade).
  • vh/vw: Viewport Height/Width (Tela inteira).

Cores

  • Hex: #FF0000 (Mais comum)
  • RGB: rgb(255, 0, 0)
  • RGBA: rgba(0, 0, 0, 0.5) (Transparência)
  • Variáveis: var(--main-color)
