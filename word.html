Formatação Ágil

Texto Essencial

  • CTRL + N Negrito
  • CTRL + I Itálico
  • CTRL + S Sublinhado
  • CTRL + SHIFT + C Copiar Formatação (Pincel)
  • CTRL + SHIFT + V Colar Formatação

Parágrafos

  • CTRL + J Justificar texto
  • CTRL + E Centralizar
  • CTRL + L Alinhar à Esquerda
  • CTRL + Space Remover formatação manual
Dica: Use SHIFT + F3 para alternar entre MAIÚSCULAS, minúsculas e Primeira Letra Maiúscula.

Navegação e Seleção

  • CTRL + Localizar (F) Abrir Painel de Navegação
  • CTRL + Ir para (G) Ir para página específica
  • CTRL + Setas Pular palavra por palavra
  • CTRL + Home Início do documento
  • F8 (2x, 3x, 4x) Expandir seleção (Palavra > Frase > Parágrafo)
  • CTRL + Click Selecionar frase inteira
  • ALT + Drag Seleção vertical (bloco de texto)

Estilos e Estrutura

Por que usar Estilos?

Nunca formate títulos manualmente (negrito/tamanho). Use Estilos para:

  • Gerar Sumário Automático.
  • Navegar pelo Painel de Navegação.
  • Mudar a fonte do documento inteiro de uma vez.

Atalhos de Títulos

  • CTRL + ALT + 1 Aplicar Título 1
  • CTRL + ALT + 2 Aplicar Título 2
  • CTRL + ALT + 3 Aplicar Título 3
  • CTRL + SHIFT + N Voltar para "Normal"

Acadêmico e Revisão

Quebras e Referências

  • CTRL + ENTER Quebra de Página (Nunca use vários Enters!)
  • CTRL + SHIFT + ENTER Quebra de Coluna
  • ALT + CTRL + F Inserir Nota de Rodapé

Revisão (Track Changes)

  • CTRL + SHIFT + E Ativar/Desativar Controlar Alterações
  • F7 Verificar Ortografia e Gramática
  • Use a aba Revisão para aceitar/rejeitar mudanças de outros autores.

Imagens e Tabelas

O segredo para imagens que não quebram o texto:

Mude a disposição do texto para "Quadrado" ou "Através". Evite "Alinhado com o texto" se quiser mover a imagem livremente.

Tabelas

  • TAB Mover para próxima célula (ou criar nova linha no final)
  • ALT + SHIFT + ↑ / ↓ Mover a linha inteira da tabela para cima/baixo

Recursos Modernos

Ditado e Leitura

  • Win + H Digitação por Voz (Ditado)
  • CTRL + ALT + Espaço Ler em Voz Alta (Immersive Reader)

Editor Microsoft

Pressione F7 ou abra o painel "Editor" para ter sugestões avançadas de clareza, concisão e linguagem formal, muito além do corretor ortográfico antigo.

Modo Escuro: Agora o Word permite deixar a página preta para descansar os olhos (Menu Exibir > Alternar Modos).
